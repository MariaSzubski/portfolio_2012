inSight
An Interactive Standardized Testing Solution

When science classes use fact memorization to prepare for standardized tests, students lose interest and fail to see the connection to the real word.

This project rethinks standardized testing though tablet and web based approach. It allows students to demonstrate conceptual knowledge and creative problem solving in an engaging, exploratory format.

inSight is my senior capstone project. This year-long project consisted of research, design, user testing and documentation.

Urban Walls:
Community Revitalization

This is a motion piece about super graphics created in Cincinnati, OH in the 1970s. This project focused on a wall design by Gordon Salchow. The visual style and transitions are based on microfilm articles used to research the project.

The first excerpt showcases Salchow's original design, revisions, and final painted wall. The second excerpt functions as the outro for the piece, summarizing the content in a newspaper layout.
